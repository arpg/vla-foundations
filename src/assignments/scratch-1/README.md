# Scratch-1: Transformer Backbone

**For full assignment instructions, requirements, and grading rubric, see:**
https://vlm-robotics.dev/course/assignments/scratch-1/

## Quick Start

```bash
# 1. Generate training data
python generate_data.py

# 2. Implement TODOs in backbone.py
# 3. Train your model
python backbone.py
```

## Files in This Directory

- `backbone.py` - Implementation template (complete the TODOs)
- `generate_data.py` - Dataset generator (provided)
- `data/` - Generated trajectory data
