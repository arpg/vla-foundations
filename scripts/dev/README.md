# Development Scripts

Local development helper scripts that are **NOT** used in CI/CD pipelines.

## Contents

- `setup_private_repo.sh` - Automated setup for private repository
- `add_github_secret.sh` - Helper to add GitHub secrets
- `deploy.sh` - Manual deployment script
- `complete-setup.sh` - Complete server setup
- `deploy-staging-dynamic.sh` - (deprecated) Dynamic staging deployment
- `review-prs.sh` - Local PR review helper

## Usage

These scripts are for local development and manual operations only. They are **not** executed by GitHub Actions.

For CI/CD scripts, see the parent `scripts/` directory.
