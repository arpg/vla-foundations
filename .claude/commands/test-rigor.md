# Test Rigor Command

Quick access to internal test runner.

## Usage

```
/test-rigor
```

## Execution

Use the Skill tool to invoke the test-rigor skill:

```
Skill: test-rigor
```

This will:
1. Prompt for assignment selection
2. Inject solutions
3. Run pytest with rigor markers
4. Generate test report
5. Reset to starter code
6. Display summary
